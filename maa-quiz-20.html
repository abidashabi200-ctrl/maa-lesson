فہمِ قرآن اکیڈمی

مَا کوئز

۲۰ سوالات — قرآن سے، آسان مثالیں
👇 سیدھا کسی بھی سوال پر جانے کے لیے نمبر دبائیں
ہر جملے میں مَا کا مطلب چنیں — "نہیں" یا "جو"۔ جواب منتخب کرتے ہی فوراً پتا چل جائے گا۔
۱
مَا وَدَّعَكَ رَبُّكَ وَمَا قَلَىٰ
( تیرے رب نے ____ تجھے چھوڑا اور ____ وہ ناراض ہوا )
۲
مَا أَغْنَىٰ عَنْهُ مَالُهُ وَمَا كَسَبَ
( اُس کا مال اور جو اُس نے کمایا، اُس کے کچھ کام ____ آیا )
۳
مَا كَذَبَ الْفُؤَادُ مَا رَأَىٰ
( دل نے جھوٹ ____ کہا جو اُس نے دیکھا )
۴
فَمَا رَبِحَت تِّجَارَتُهُمْ
( تو اُن کی تجارت نے نفع ____ دیا )
۵
مَا لَكُم مِّنْ إِلَٰهٍ غَيْرُهُ
( تمہارے لیے اُس کے سوا کوئی معبود ____ )
۶
مَا جَعَلَ اللَّهُ مِن بَحِيرَةٍ وَلَا سَائِبَةٍ
( اللہ نے ____ بحیرہ بنائی، نہ سائبہ )
۷
مَا كُنَّا مُهْلِكِي الْقُرَىٰ إِلَّا وَأَهْلُهَا ظَالِمُونَ
( ہم بستیوں کو ہلاک کرنے والے ____ تھے مگر یہ کہ وہاں کے لوگ ظالم ہوں )
۸
وَمَا رَبُّكَ بِظَلَّامٍ لِّلْعَبِيدِ
( اور تیرا رب بندوں پر ظلم کرنے والا ____ )
۹
وَمَا كَانَ صَلَاتُهُمْ عِندَ الْبَيْتِ إِلَّا مُكَاءً وَتَصْدِيَةً
( اور اُن کی نماز بیت اللہ کے پاس ____ تھی مگر سیٹی اور تالی بجانا )
۱۰
وَمَا أَنَا عَلَيْكُم بِوَكِيلٍ
( اور میں تم پر نگہبان ____ ہوں )
۱۱
وَمَا أُوتِيتُم مِّنَ الْعِلْمِ إِلَّا قَلِيلًا
( اور تمہیں علم میں سے تھوڑا ہی دیا گیا ہے — یعنی زیادہ ____ دیا گیا )
۱۲
يَعْلَمُ مَا بَيْنَ أَيْدِيهِمْ وَمَا خَلْفَهُمْ
( وہ جانتا ہے ____ اُن کے آگے ہے اور ____ اُن کے پیچھے ہے )
۱۳
لَهُ مَا فِي السَّمَاوَاتِ وَمَا فِي الْأَرْضِ
( اُسی کا ہے ____ آسمانوں میں ہے اور ____ زمین میں ہے )
۱۴
خُذُوا مَا آتَيْنَاكُم بِقُوَّةٍ
( ____ ہم نے تمہیں دیا ہے، اُسے مضبوطی سے پکڑ لو )
۱۵
وَاذْكُرُوا مَا فِيهِ
( اور یاد رکھو ____ اُس میں ہے )
۱۶
لَهَا مَا كَسَبَتْ وَعَلَيْهَا مَا اكْتَسَبَتْ
( اُس کے لیے ہے ____ اُس نے کمایا، اور اُس پر ہے ____ اُس نے کمایا )
۱۷
رَبَّنَا آتِنَا مَا وَعَدتَّنَا عَلَىٰ رُسُلِكَ
( اے ہمارے رب، ہمیں وہ عطا فرما ____ کا تُو نے اپنے رسولوں کے ذریعے وعدہ کیا )
۱۸
وَمَا تَشَاءُونَ إِلَّا أَن يَشَاءَ اللَّهُ
( تم کچھ ____ چاہ سکتے مگر یہ کہ اللہ چاہے )
۱۹
وَلَهُم مَّا يَدَّعُونَ
( اور اُن کے لیے ہے ____ وہ مانگیں گے )
۲۰
وَمَا كَانَ لِنَفْسٍ أَن تَمُوتَ إِلَّا بِإِذْنِ اللَّهِ
( کسی جان کے لیے ممکن ____ کہ مرے مگر اللہ کے حکم سے )
